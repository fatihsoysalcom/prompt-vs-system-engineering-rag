# Prompt vs System Engineering RAG

This example illustrates the shift from basic prompt engineering to a more comprehensive system engineering approach for enterprise AI. It demonstrates how a simple prompt might yield generic results, while incorporating a basic Retrieval Augmented Generation (RAG) system provides the LLM with specific context, leading to more accurate and relevant answers.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
